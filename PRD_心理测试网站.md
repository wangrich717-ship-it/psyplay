# PRD：心理测试一站式工具网站

**项目名称：** PsyPlay（玩心理）  
**版本：** v1.0  
**文档日期：** 2026年3月  
**技术约束：** 纯前端（HTML + CSS + JS），无后端，无登录，无数据存储，无AI接口，可Vibe Coding实现

---

## 一、产品概述

### 1.1 产品定位

一个好玩、有趣、轻量的心理测试聚合网站。用户进入即玩，玩完即看结果，不注册、不登录、不存数据，轻松解压。

测试类型多样：有文字题、有图片选择、有声音交互、有动画计时等，增加沉浸感和娱乐性。

### 1.2 目标用户

- 闲来无事想娱乐的普通用户
- 对心理测试好奇的年轻人（18-35岁）
- 想了解自己性格/情绪/偏好的用户

### 1.3 核心原则

1. **好玩优先**：测试体验要有趣，不学术，不枯燥
2. **结果要爽**：每个测试结果要有具体描述、有画面感、有代入感
3. **交互多样**：混合文字/图片/声音/动画等多种形式
4. **即开即用**：纯静态，打开就能玩

## 三、全局设计规范

### 3.1 视觉风格

- 主色调：深紫 `#2D1B69` + 浅奶白 `#FDF8F0` + 荧光橙 `#FF6B35`
- 字体：系统中文字体（思源黑体/苹方/微软雅黑 fallback）
- 卡片设计：圆角 16px，轻阴影，hover有微动画
- 进度条：每个测试顶部显示当前题目进度

### 3.2 交互规范

- 每道题选完自动跳下一题（无需点"下一题"）
- 最后一题后展示加载动画（1.5秒"分析中…"假loading，增强仪式感）
- 结果页：标题 + 类型名称 + 2-3段描述 + 一句"金句" + 分享按钮（复制文字）
- 所有测试支持「重新测试」按钮

### 3.3 技术实现

- 单页应用：用 `section` 切换隐藏/显示模拟路由
- 状态管理：JS对象存储当前测试进度和答案分数
- 声音：`<audio>` 标签 + Web Audio API 生成音调（无需音频文件）
- 图片：用CSS绘制的纯色块/SVG图形代替真实图片（避免资源依赖）
- 分享：`navigator.clipboard.writeText()` 复制结果文字

### 测试3：你的情绪「音调」

**类型：** 声音交互测试  
**题目数：** 6题  
**用时：** 约2分钟  
**标语：** "闭上眼睛，听，你的身体比你更诚实"

#### 玩法说明

用Web Audio API实时生成不同频率/节奏的音调（无需音频文件）。用户点击播放按钮，听完后选择"这个声音让你感觉到什么"。通过对不同声音的情绪反应，分析情绪敏感度。

#### 技术实现要点

```javascript
// 用 AudioContext 生成纯音调，不依赖外部文件
const AudioContext = window.AudioContext || window.webkitAudioContext;
const ctx = new AudioContext();

function playTone(frequency, duration, type='sine') {
  const oscillator = ctx.createOscillator();
  const gainNode = ctx.createGain();
  oscillator.connect(gainNode);
  gainNode.connect(ctx.destination);
  oscillator.frequency.value = frequency;
  oscillator.type = type; // sine/square/sawtooth/triangle
  gainNode.gain.setValueAtTime(0.3, ctx.currentTime);
  gainNode.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + duration);
  oscillator.start(ctx.currentTime);
  oscillator.stop(ctx.currentTime + duration);
}
```

#### 题目设计

| 题号 | 音调描述 | 技术参数 | 听完你感觉到？ |
|------|---------|---------|-------------|
| 1 | 低沉持续的嗡鸣 | 80Hz，sine，3秒 | A.压迫 B.稳定 C.想睡 D.不舒服 |
| 2 | 高频短促的叮声 | 1200Hz，triangle，0.5秒×3 | A.警觉 B.清醒 C.烦躁 D.兴奋 |
| 3 | 中频缓慢起伏 | 440Hz，sine，渐强渐弱，4秒 | A.平静 B.忧伤 C.期待 D.无感 |
| 4 | 不规则节奏的敲击 | 200Hz，square，不规律间隔 | A.焦虑 B.有趣 C.心跳加速 D.想关掉 |
| 5 | 和谐双音调 | 264Hz+330Hz同时，sine，3秒 | A.温暖 B.诡异 C.放松 D.想哭 |
| 6 | 完全的静音（0.5秒后提示"这就是题目"） | 无声，展示"静默本身是一种声音" | A.舒适 B.难受 C.好奇 D.空洞 |

#### 结果类型（4种）

| 类型 | 名称 | 描述 |
|------|------|------|
| 高敏感型 | 🎵 你是「情绪高保真」 | 你对外界刺激的感知极其细腻。声音、语气、空气中的细微变化，你都能感受到。这是天赋，也是消耗。 |
| 防御型 | 🔇 你是「情绪降噪模式」 | 你已经学会了自动屏蔽一部分感受。不是你冷漠，是你曾经感受得太多了，你的神经系统自动保护你。 |
| 矛盾型 | 🔀 你是「情绪混音台」 | 你的感受常常是矛盾叠加的——同一件事让你同时感到开心和难过，同时感到渴望和抗拒。 |
| 平衡型 | 🎼 你是「情绪调音师」 | 你有稳定的内在基准线，不容易被外界的声音带乱节奏。你知道什么时候该让自己感受，什么时候该关上门。 |

### 测试6：你的「崩溃临界值」是多少？

**类型：** 场景问答，有动态进度条  
**题目数：** 10题  
**用时：** 约3分钟  
**标语：** "每个人都有一条崩溃线，你的在哪里？"

#### 玩法说明

题目描述真实的日常压力场景，选项是不同程度的反应。答题过程中，顶部有一根"压力指数条"，每答一题会根据选项动态变化（视觉反馈）。最终分数决定「崩溃临界值」。

#### 题目设计（选取5题示意）

| 题号 | 场景 | 选项（代表不同耐压程度） |
|------|------|----------------------|
| 1 | 你的外卖比预计晚了一个小时，你会？ | A.完全没问题 B.有点烦但忍了 C.发差评 D.开始想今天哪里不对劲 |
| 2 | 工作群突然@你说一件你以为早解决的事，你的反应是？ | A.淡定回复 B.叹口气 C.心跳加速 D.直接摆烂一下 |
| 3 | 你期待很久的计划被临时取消，你会？ | A.好的没事 B.有点失落但调整 C.情绪低落一整天 D.开始对所有事失去期待 |
| 4 | 你昨晚没睡好，今天还有一堆事要做，你最可能？ | A.照常运转 B.咖啡续命 C.效率减半但撑着 D.开始觉得活着好累 |
| 5 | 有人当众说了你一句你觉得不公平的话，你会？ | A.当场反驳 B.忍住但记住了 C.表情管理失败 D.接下来几天反复想这件事 |

（其余5题类似，覆盖：工作/关系/身体/意外/自我期待场景）

#### 结果类型（4种）

| 分数段 | 类型 | 名称 | 描述 |
|------|------|------|------|
| 80-100 | 高耐压型 | 🪨「你是一块石头，但石头也会裂」 | 你的崩溃临界值很高，别人觉得你什么都扛得住。但注意：你不是真的不崩溃，只是你的崩溃来得更晚、更大、更没有预兆。 |
| 55-79 | 稳定型 | 🌊「你像水，会晃动但不会溢出」 | 你对压力有感知，也有消化能力。你不会假装没事，但你通常能找到出口。这是健康的耐压方式。 |
| 30-54 | 敏感型 | 🕯️「你是一根蜡烛，风大了会灭」 | 你的感受很细腻，对压力的反应比别人快。这不是缺点——敏感意味着你活得很深，只是需要更好地学会给自己充电。 |
| 0-29 | 临界型 | 🧨「你现在站在自己的临界线上」 | 你已经很久没有好好休息了对不对？你的身体和内心都在发出信号——不是你脆弱，是你太久没被照顾了，包括被你自己照顾。 |

### 测试7：你用什么方式爱自己？

**类型：** 行为频率量表（滑块/单选）  
**题目数：** 10题  
**用时：** 约3分钟  
**标语：** "爱自己不是买买买，是这些微小的瞬间"

#### 玩法说明

每题用滑块表示行为频率（从不/偶尔/经常/总是），最终生成雷达图（用Canvas绘制），显示用户在5个维度的自我照料程度。

#### 五个维度

1. **身体照料**（吃好睡好运动）
2. **情绪允许**（允许自己悲伤/愤怒/休息）
3. **边界设定**（会拒绝、会说不）
4. **自我对话**（内心独白是温柔还是苛刻）
5. **快乐供给**（给自己提供纯粹的快乐和娱乐）

#### 题目设计（各维度2题示意）

| 维度 | 题目 |
|------|------|
| 身体 | 我会因为「好吃」而选择某样食物，而不只是「健康」；我的睡眠时间是我主动保护的，不是「剩下来」的 |
| 情绪 | 我允许自己在没有理由的情况下感到悲伤；我有哭的空间——不是憋着，是真的哭出来 |
| 边界 | 我能对额外的要求说"我现在没有精力"；别人失望时，我不会立刻放弃自己的需求 |
| 对话 | 我犯错后给自己说的话，和我安慰朋友说的差不多；我不会用苛刻的标准反复审判自己 |
| 快乐 | 我有纯粹为了好玩而做的事情，不需要它有任何意义；我允许自己浪费时间 |

#### 结果展示

- 用Canvas绘制五边形雷达图
- 每个维度给出1句针对性建议
- 总结：「你的自我照料模式是……」

### 测试8：三秒决定——你的本能防御机制

**类型：** 限时图片选择（强制3秒内必须点击）  
**题目数：** 8题  
**用时：** 约1.5分钟  
**标语：** "不要想，三秒内点击——你的本能是最诚实的"

#### 玩法说明

每题展示4个选项（CSS图示或词语大字卡），倒计时3秒，时间到自动选"跳过"（等同于某种答案），逼迫用户用直觉而非理智作答。

#### 技术要点

```javascript
let timer;
function startCountdown(questionIndex) {
  let count = 3;
  const bar = document.getElementById('timer-bar');
  timer = setInterval(() => {
    count -= 0.1;
    bar.style.width = (count / 3 * 100) + '%';
    if (count <= 0) {
      clearInterval(timer);
      autoSelect(questionIndex); // 超时自动选"逃避"选项
    }
  }, 100);
}
```

#### 题目设计

| 题号 | 题干 | 选项（4个大字卡，用颜色区分） |
|------|------|--------------------------|
| 1 | 有人对你发火——你第一反应是？ | 道歉 / 反击 / 解释 / 消失 |
| 2 | 被人冷落——你最先感受到的是？ | 愤怒 / 难过 / 自我怀疑 / 算了 |
| 3 | 你搞砸了一件事——你对自己说？ | 没事的 / 都是我的错 / 找原因 / 沉默 |
| 4 | 关系中遇到冲突——你本能是？ | 说清楚 / 忍下来 / 冷战 / 离开 |
| 5 | 别人比你优秀——你感到？ | 羡慕 / 嫉妒 / 努力 / 无所谓（但其实有） |
| 6 | 很久没人联系你——你会？ | 主动联系 / 等待 / 觉得被遗弃 / 告诉自己不在乎 |
| 7 | 面对不确定的未来——你本能是？ | 计划 / 担心 / 顺其自然 / 不去想 |
| 8 | 有人说"我们谈谈"——你第一感觉是？ | 好啊 / 我做错了什么 / 紧张 / 想逃跑 |

#### 结果类型（4种防御机制）

| 类型 | 名称 | 描述 |
|------|------|------|
| 战斗型 | ⚔️「你的本能是迎战」 | 你在本能上倾向于正面应对威胁。你不容易被吓跑，但有时候你的"战斗"模式会在没有真正威胁的场合也自动启动。 |
| 逃跑型 | 🏃「你的本能是先离开现场」 | 你的自我保护方式是撤退。这是一种高效的应激反应，但长期使用会让你错过很多本可以解决的关系和机会。 |
| 僵住型 | 🫧「你的本能是冻结」 | 当面对威胁时，你的系统会暂时"卡机"。你不是不想反应，是你的神经系统在保护你——通过暂停来争取时间。 |
| 取悦型 | 🤲「你的本能是先照顾别人的感受」 | 你在威胁出现时，第一反应是让对方舒服。这来自你深层的善意，也来自早年习得的"让别人满意=我安全"的经验。 |

### 测试9：你的内心独白是什么声音？

**类型：** 声音情绪匹配（Web Audio API生成多种音效）  
**题目数：** 6题  
**用时：** 约2分钟  
**标语：** "如果你的内心独白是一种声音，它是什么？"

#### 玩法说明

每题描述一个心理场景，用户先选择"当时脑子里的声音类型"（提供4种声音选项，点击可试听），选完后从文字选项中选择"这个声音传达的情绪"。双重选择综合分析内心对话模式。

#### 声音类型库（Web Audio API实现）

| 声音名称 | 技术参数 | 描述 |
|---------|---------|------|
| 回声低语 | 80Hz sine + reverb模拟 | 低沉、遥远、像在洞里说话 |
| 尖锐提示音 | 800Hz triangle, short | 急促、警告感 |
| 平静持续音 | 220Hz sine, steady | 稳定、平和 |
| 断断续续杂音 | random freq square | 混乱、焦虑感 |

#### 题目设计

| 题号 | 场景 | 你选哪种声音？ |
|------|------|-------------|
| 1 | 你刚发出一条消息，对方迟迟没有回 | 以上4种声音 |
| 2 | 你独自一人在家，突然感到莫名难过 | 以上4种声音 |
| 3 | 你刚完成了一件事，回头看自己的表现 | 以上4种声音 |
| 4 | 你想开口说一件事，但最终没说 | 以上4种声音 |
| 5 | 你看到别人过得很好，想到自己 | 以上4种声音 |
| 6 | 你睡前躺着，大脑开始活动 | 以上4种声音 |

#### 结果类型（4种内心独白模式）

| 类型 | 名称 | 描述 |
|------|------|------|
| 批评者 | 🔍「你的内心住着一个严格的批评者」 | 你脑子里有一个声音，专门负责审判你——你做得够好吗？你说的话正确吗？你值得这些吗？这个声音想保护你，但用的方式太苛刻了。 |
| 担忧者 | 🌀「你的内心住着一个总在预警的担忧者」 | 你的内心独白常常在预演最坏的情况。你不是悲观，你只是被训练成了"提前防御"的模式。 |
| 旁观者 | 👁️「你的内心住着一个冷静的旁观者」 | 你能从某种距离观察自己的感受，不完全被它淹没。这是一种心理成熟度，也可能是一种轻微的解离——你有时候不确定自己"在不在"。 |
| 鼓励者 | 🌱「你的内心住着一个不放弃你的鼓励者」 | 哪怕你经历了很多，你的内心独白里依然有一部分在说"没事的，可以的"。这很珍贵，好好保护它。 |

### 测试10：你的「关系依赖地图」

**类型：** 多维度文字测试，结果可视化（Canvas绘制地图）  
**题目数：** 12题  
**用时：** 约4分钟  
**标语：** "你和别人的距离，藏着你的全部秘密"

#### 玩法说明

测试用户的依恋风格（安全/焦虑/回避/混乱）+ 关系中的核心需求。结果展示一张可视化"关系地图"：一个同心圆，不同的关系类型被标记在不同距离的圆环上，配合文字分析。

#### 题目设计（按维度分组）

**A. 亲密感需求（4题）**

| 题号 | 题目 | 选项 |
|------|------|------|
| 1 | 在关系中，你更害怕哪个？ | A.太近（失去自我）B.太远（被遗弃） |
| 2 | 有人突然对你表示强烈的喜欢，你的直觉是？ | A.感动 B.警惕 C.不知所措 D.开始期待和担心同时出现 |
| 3 | 你在关系里更容易做哪件事？ | A.主动靠近 B.等待被靠近 C.靠近又退缩 D.保持稳定距离 |
| 4 | 分别时（哪怕只是下班），你通常感觉？ | A.如释重负 B.有点空 C.没什么感觉 D.开始担心对方还在不在 |

**B. 信任建立（4题）**

| 题号 | 题目 |
|------|------|
| 5 | 你需要多久才能真正信任一个人？ |
| 6 | 有人让你失望过一次，你会？ |
| 7 | 你愿意让别人知道你软弱的一面吗？ |
| 8 | 你相信大多数人是真心对你的吗？ |

**C. 冲突处理（4题）**

| 题号 | 题目 |
|------|------|
| 9 | 关系中出现问题，你会主动提出来吗？ |
| 10 | 对方沉默时，你通常假设是？ |
| 11 | 争吵后，你更容易做的是？ |
| 12 | 你允许关系中存在未解决的部分吗？ |

#### 结果类型（4种依恋风格）

| 类型 | 名称 | 描述 | 地图特征 |
|------|------|------|---------|
| 安全型 | 🏡「你的关系地图是一个温暖的营地」 | 你能靠近，也能独处。你不会因为关系而失去自己，也不会为了保护自己而让关系冻死。 | 各圈层分布均匀，中心圈标记"自我" |
| 焦虑型 | 🧲「你的关系地图是一块磁铁」 | 你渴望亲密到有时候会把人吓跑。不是你粘，是你内心某个孩子太需要被确认"我不会被丢下"。 | 内圈过密，外圈几乎空白 |
| 回避型 | 🏝️「你的关系地图是一座孤岛」 | 你给自己造了一个完好的岛，岛上的生活井井有条。但你偶尔会站在岸边，想象对岸是什么样子。 | 各人都在最外圈，中心只有"自我" |
| 混乱型 | 🌪️「你的关系地图是一场风暴」 | 你同时渴望亲密和恐惧亲密。你靠近又后退，被接受又不敢相信。这通常来自早年的关系经历——不是你的错，是你学到的。 | 人物位置随机分散，没有规律 |
